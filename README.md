
# 2021年10月10日-11日基礎逆向工程

```
教育部先進資通安全實務人才培育計畫
校園資安深耕營Advanced Happy CyberSecurity Day

課程名稱：基礎逆向工程
課程說明：本次課程將從零開始帶領學員了解逆向工程、組合語言等相關的基礎，讓初學者更容易切入逆向工程。
課程內容包含動態分析、靜態分析並且分析PE格式(Windows EXE)與Linux ELF格式。
本課程將使用下列工具Ghidra、PEBear、x64dbg、gdb進行測試。

課程大綱：
-程式在底層是如何運行的
-基礎x86組合語言指令
-逆向工程動/靜態分析方式
-Windows exe 逆向工程
-Linux ELF 逆向工程
-加快逆向工程的方式
```
